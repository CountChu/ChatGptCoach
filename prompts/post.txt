I define the commands and describe them in the "//"" comments as below.

//
// The post-begin command informs you that the title of the document to be posted is TITLE.
//

post-begin: TITLE 

//
// The post-pages command means that I post the TITLE document many pages. 
// The N in "Page N" means a page number. E.g., Page 1
// The N+1 in "Page N+1" means the next page number. E.g., Page2
// The CONTENT means the content of the page.
// Please just accept them and don't response them.
//

post-pages:

Page N

CONTENT

Page N+1

CONTENT

...

/
// An example of the first post-pages command:
//

post-pages:

Page 1

HKG18-402: Secure Key Services in 
OP-TEE 
Etienne Carrière, STMicroelectronics 

Page 2
